* php artisan serve

crear modelos

* php artisan make:model <nombre>

crear controllers

*php artisan controller:make [Name]Controller


* migraciones

crear migracion 

* php artisan make:migration <nombre_de_la_tabla>


ejecutar migracion 

* php artisan migrate

modificar tablas

* php artisan make:migration add price_to_product --table=product



revertir migraciones

* php artisan migrate:rollback


revertir todas las migraciones

* php artisan migrate:reset


claves foráneas

* php artisan make:migration add_category_id_to_product_table --table=nombre_tabla




seeders

crear seeder

* php artisan make:seeder CategoryTableSeeder

cargar seed

* php artisab db:seed



Resetear toda la base de datos:


* php artisan migrate:fresh --seed
